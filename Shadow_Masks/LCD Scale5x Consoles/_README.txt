A set of LCD handheld Shadow Masks for use with console cores.

This settings below are required to get square 5x5 pixels, that will align to
the provided Shadow Masks.

Note: resolution changes by the Sega Genesis or Sega CD core may offset the
Vertical Crop in a way that does not correctly align.
